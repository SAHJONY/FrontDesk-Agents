## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/frontdesk-agents.git

# Navigate to project
cd frontdesk-agents

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to Vercel
vercel deploy
```
