## Core Architecture

### Technology Stack
- **Frontend:** Next.js 14, React 18, TypeScript, TailwindCSS
- **Backend:** FastAPI (Python 3.11), Node.js
- **AI Models:** NVIDIA NIM (Qwen3.5-397B), GPT-4o, Claude 3.5
- **Voice:** LiveKit (real-time), Bland.ai (phone), Twilio (SMS)
- **Database:** PostgreSQL (Supabase), Redis (cache), Pinecone (vectors)
- **Languages:** 200+ via NLLB, Whisper, Google Translate
- **Deployment:** Vercel (frontend), Railway (backend), LiveKit Cloud

### Global Coverage
- **Languages:** 200+ (all major world languages)
- **Dialects:** Regional variants supported
- **Timezones:** Automatic detection and conversion
- **Currencies:** 150+ with real-time conversion
- **Compliance:** GDPR, HIPAA, SOC2 ready
